🎯 دليل التطوير المهني (نسخة 2025 التفاعلية)
واجهة ويب متطورة وتفاعلية تقدم منهجية شاملة للتطوير المهني بناءً على أربعة محاور استراتيجية. يتميز المشروع بتصميم عصري يجمع بين الأناقة (Dark & Gold Aesthetic) والتجربة البرمجية السلسة.

🚀 نظرة عامة
تم تصميم هذا الدليل ليكون مرجعاً بصرياً وتفاعلياً للمحترفين الطموحين، حيث يستعرض البيانات والمفاهيم المهنية باستخدام رسوم متحركة (Animations) ومخططات بيانية ذكية تعتمد على تقنيات الويب الحديثة دون الحاجة لمكتبات خارجية ثقيلة.

💎 المحاور الأربعة (Pillars)
يستعرض المشروع المحتوى عبر أربعة أقسام رئيسية:

بناء المهارات (Skills): هرم المهارات (الصلبة، الناعمة، الذكاء الاصطناعي، والحكمة) وقاعدة 70-20-10 للتعلّم.

الشبكة المهنية (Networking): استراتيجيات بناء العلاقات وجدول الاستخدام الأمثل للمنصات الرقمية.

التطوير الشخصي (Personal Dev): خط زمني للعادات اليومية، مصفوفة إيزنهاور للأولويات، وقاعدة 80/20.

التخطيط الوظيفي (Career Planning): الآفاق الزمنية الثلاثة وإطار أهداف الـ OKRs المهني.

🛠️ المزايا التقنية (Tech Stack)
اللغات: HTML5, CSS3 (Variables, Grid, Flexbox), Vanilla JavaScript.

التفاعلية: استخدام Intersection Observer API لتشغيل العدادات والرسوم المتحركة عند التمرير.

التصميم البصري: تأثير "Noise Overlay" باستخدام SVG، وتدرجات لونية مخصصة، وتصميم متجاوب بالكامل (Responsive Design).

تحليل البيانات: نظام تتبع متطور مدمج (Google Analytics 4) مخصص لمراقبة عمق التمرير ووقت البقاء في الصفحة.

🔒 أمن المعلومات وحماية المحتوى
تم تزويد المشروع بطبقة أمان برمجية قوية تشمل:

سياسة أمن المحتوى (CSP): لمنع حقن الأكواد الضارة.

حماية الواجهة: منع النقر بيمين الفأرة (Right-click)، منع نسخ النصوص، وتعطيل اختصارات لوحة المفاتيح الحساسة (F12, Ctrl+U).

Frame-busting: منع تضمين الصفحة داخل iframe لحماية الموقع من هجمات Clickjacking.

📂 هيكلة الملف
المشروع عبارة عن صفحة واحدة مستقلة (Single File Component)، مما يسهل عملية النقل والتشغيل:

Styles: نظام CSS مخصص يعتمد على المتغيرات (Variables) لسهولة تعديل الهوية البصرية.

Security Scripts: كتلة برمجية مخصصة لحماية المحتوى في أعلى الملف.

Analytics Scripts: إعدادات GA4 المتقدمة مع أحداث مخصصة (Custom Events).

Main Content: الهيكل التنظيمي للأقسام الأربعة.

⚙️ كيفية التخصيص
يمكنك تعديل البيانات بسهولة عبر فتح الملف وتعديل القيم التالية:

الأرقام والعدادات: قم بتغيير data-target في وسوم العدادات.

أهداف الـ OKR: عدّل قيم data-w في قسم التخطيط الوظيفي لتغيير أشرطة التقدم.

الهوية البصرية: عدّل الألوان في قسم :root تحت مسميات --gold و --teal و --ink.
